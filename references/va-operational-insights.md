## 四、评论战术库

### 4.1 三种战术定义

| 战术 | 比例 | 适用场景 | 效果 |
|------|------|---------|------|
| 专业钩子型 | 30% | 帖主有明确疑问，我方可提供关键信息 | 引发对话，子回复率高 |
| 专业补充型 | 60% | 帖子讨论某项目，我方补充更多信息 | 展示专业度，品牌曝光 |
| 专业延伸型 | 10% | 帖子触及相关话题，引导至医美决策 | 教育用户，间接引流 |

### 4.2 高效评论结构

**onda 类帖子**：
```
[判断：脂肪型 vs 皮肤松弛型] → [项目选择建议] → [VA 价格(TWD)] → [引导咨询]
```

**音波对比类帖子**：
```
[各机器特点对比] → [适合人群说明] → [VA 有哪些机器] → [诊所引导]
```

**赴韩计划类帖子**：
```
[时效性响应：5月还来得及] → [需求匹配：VA 有 X/Y/Z] → [组合价格(TWD)] → [预约引导]
```

**价格对比类帖子**：
```
[台湾价格 vs 韩国价格差距] → [VA 具体项目(TWD)] → [中文接待强调]
```
